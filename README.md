# study-buddy

### GET STARTED

1. First go into the back end directory:

```cd back-end```

2. Run the backend server:

```npm run start```

OR

```npm run dev``` (using nodemon!)

3. Then CD into the front-end
```cd ../```
```cd study-buddy-front-end/```

4. Run the React App

```npm run start``` 

or

```npm start```

### The app will be on http://localhost:3000
